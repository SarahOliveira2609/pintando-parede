# 🖌️ Pintando Parede

Este projeto foi desenvolvido para praticar conceitos básicos de entrada de dados e cálculos matemáticos em Python.

## 📝 Sobre o Projeto
O programa solicita ao usuário as dimensões de uma parede (largura e altura) e realiza o cálculo automático da área total e da quantidade de litros de tinta necessária, baseando-se no rendimento padrão de 1 litro para cada 2m².
